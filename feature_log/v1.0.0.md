# FlightScanner v1.0 功能迭代记录

**发布日期**: 2026-02-28

**版本**: v1.0.0

## 版本概述

FlightScanner v1.0 是一个重要的版本更新，从 CLI-only MVP 升级为完整的 Web 应用，引入了航线监控管理、自动化价格追踪和可视化仪表板。

## 主要功能

### 1. 数据库模型升级

#### 新增 Route 模型

在 `src/flightscanner/models/database.py` 中新增航线监控配置表：

```python
class Route(Base):
    """航线监控配置表"""
    __tablename__ = "routes"

    id = Column(Integer, primary_key=True)
    origin = Column(String(50), nullable=False)          # 出发城市
    destination = Column(String(50), nullable=False)     # 到达城市
    target_date = Column(Date, nullable=False)           # 目标日期
    target_price = Column(Numeric(10, 2), nullable=False) # 目标价格
    is_active = Column(Integer, default=1)               # 监控状态
    created_at = Column(DateTime, default=utcnow)
    updated_at = Column(DateTime, onupdate=utcnow)
```

**特性**：
- 支持激活/暂停监控（is_active 字段）
- 自动时间戳管理
- 级联删除关联的价格历史记录

#### PriceHistory 模型扩展

在 `PriceHistory` 表中新增 `route_id` 外键：

```python
route_id = Column(Integer, ForeignKey("routes.id"), nullable=True, index=True)
```

**设计考虑**：
- `route_id` 设为可空，保持与现有数据的兼容性
- 新增索引 `ix_price_history_route_scraped` 提升查询性能

### 2. 服务层架构

#### RouteService 业务服务

新增 `src/flightscanner/core/services/route_service.py`：

**核心功能**：
- `add_route()` - 添加航线监控（去重验证）
- `get_all_routes()` - 获取所有航线（含最新价格统计）
- `get_active_routes()` - 获取活跃监控航线
- `toggle_route_status()` - 切换监控状态
- `delete_route()` - 删除航线
- `get_route_price_history()` - 查询航线价格历史
- `save_price_for_route()` - 保存价格快照并关联航线

**数据传输对象**：

```python
@dataclass
class RouteWithLatestPrice:
    """航线及最新价格信息"""
    id: int
    origin: str
    destination: str
    target_date: date
    target_price: Decimal
    is_active: bool
    created_at: datetime
    latest_price: Optional[Decimal]
    latest_scraped_at: Optional[datetime]
    price_count: int
```

**查询优化**：
- 使用子查询获取每条航线的最新价格
- 单次 JOIN 查询获取完整数据
- 添加复合索引优化常用查询
