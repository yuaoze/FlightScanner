# FlightScanner v1.5.0 功能更新

**发布日期**: 2026-04-30

**版本**: v1.5.0

---

## 版本概述

v1.5.0 上线 **周末灵感雷达（Weekend Radar）**：自动扫描未来 N 个周末从上海出发的周五晚 → 周日晚往返机票，按历史均价偏离度排序，用 AI 生成行程文案，支持一键"锁定价格并监控"无缝转为精准跟踪路线。

该功能依赖 v1.4.2 打通的国内往返采集链路和新增的回程时间窗过滤。

---

## 核心功能

### 扫描逻辑

**时间窗对齐**（[`scanner.py`](../src/flightscanner/weekend_radar/scanner.py)）

| 阶段 | 时间窗 |
|---|---|
| 去程起飞 | 周五 ≥ 19:00 |
| 去程落地 | 周六 ≤ 02:00 |
| 回程起飞 | 周日 18:00 – 23:59 |
| 回程落地 | 周一 ≤ 02:00 |

符合该时段的航班组合才视为"真·周末短途"。

### 目的地池

[`destinations.py`](../src/flightscanner/weekend_radar/destinations.py) 内置国内 + 国际候选：

- **国内**：北京、广州、深圳、成都、重庆、西安、昆明、厦门、杭州、青岛、哈尔滨、三亚 等
- **国际**：东京、大阪、首尔、曼谷、清迈、新加坡、吉隆坡、香港、澳门、台北、河内、胡志明 等
- 每目的地配签证/签注信息、地标图片映射

### UI 入口

[`ui/app.py`](../ui/app.py) 顶部 tab 栏新增 **🌍 周末灵感雷达** 标签（与 📋 当前监控 / ⏱️ 历史记录 并列）。打开后：

1. 选择扫描哪几个周末（默认未来 4 个）
2. 点击「🔎 立即扫描」触发批量采集
3. 结果按"击败历史均价百分比"排序，每张卡片展示：
   - 背景图（城市地标）
   - 出发/回程航班号、时刻、机场
   - 往返总价 + 历史均价对比
   - AI 行程文案（DeepSeek 生成，失败降级规则文案）
   - 「🌙 红眼预警」「🔥 超值」「✈️ 直飞」等标签
   - 「❤️ 锁定价格并监控」按钮

### "锁定价格并监控"流程

[`_add_to_monitoring()`](../ui/components/weekend_radar.py) 接收一条 `WeekendDeal`，三步完成：

1. **新建路线**：
   - `trip_type="roundtrip"`，目标价 = 当前往返总价 × 0.95
   - `dep_time_from="19:00"`（去程窗口）
   - `ret_dep_time_from="18:00"`、`ret_dep_time_to="23:59"`（回程窗口，依赖 v1.4.2 新增的回程时间窗字段）
2. **立即采集一次**：调用 `trigger_immediate_scrape(new_route_id)`，带 spinner 完整跑一次 Qunar + Ctrip 采集管道，避免用户等到下一个调度周期才看见数据
3. **`st.rerun()`** 触发页面重渲染，新路线带着首次采集结果一起出现在当前监控标签里

### 缓存

[`models/database.py`](../src/flightscanner/models/database.py) 新增 `weekend_radar_cache` 表，存储一次扫描产出的所有推荐组合（原始价格、历史均价、击败百分比、AI 简报 JSON）。同目的地同日期对不重复扫描，有效期为当日。

## 技术实现要点

### 价格序列按天聚合（`analyzers/deepseek_analyzer.py`）

周末雷达的 AI 简报生成对**历史均价趋势**敏感。原先 `DeepSeekBriefingAnalyzer` 把价格序列直接取最近 30 条，同一天内多次采集造成价格点堆积、趋势被噪声淹没。改为按日期聚合取最低价：

```python
daily_min: dict[date_str, (min_price, time_str, source)]
for fp in price_history:
    day = fp.scraped_at.date()
    if day not in daily_min or fp.price < daily_min[day][0]:
        daily_min[day] = (fp.price, ..., fp.source)
price_series = sorted(daily_min.items())[-30:]   # 最近 30 天的每日最低价
```

更稳定、更线性、覆盖更长的时间跨度。单条路线的 AI 简报同样受益。

### 周末日期计算

[`get_upcoming_weekends(n)`](../src/flightscanner/weekend_radar/scanner.py) 返回未来 n 个 `(周五, 周日)` 元组：

- 今天周一–周四：本周五起算
- 今天已是周五：本周五仍可（若航班还未出发）
- 今天周六/周日：从下周五起算

### 价格对比基准

`_compute_historical_avg()` 查 `price_history` 表同目的地过去 N 天的最低价均值作为基准。`beat_pct = int((avg - total_price) / avg * 100)`，≥ 20% 的标"🔥 超值"。

### AI 简报降级

[`brief_generator.py`](../src/flightscanner/weekend_radar/brief_generator.py) 调用 DeepSeek 生成 200 字行程文案（城市亮点 + 玩法建议 + 航班亮点）；API 失败时按目的地 + 价格档位返回规则模板。`_add_to_monitoring` 不依赖该字段，降级不影响核心功能。
